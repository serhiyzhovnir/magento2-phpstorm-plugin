The Listing (Grid) component configuration with filtering, pagination, sorting, and other features.

Template's predefined variables:
${NAME}   The name of the component.
${ID_FIELD_NAME}   The identifier field name.
${ACL}   The name of ACL resource that allows to restrict an access to the listing view.
${PROVIDER_CLASS}   The data provider class name.
${TOOLBAR}   Adding the Toolbar component configuration.
${BOOKMARKS}   Adding the Bookmarks component configuration.
${COLUMNS_CONTROLS}   Adding the Columns Controls component configuration.
${FULLTEXT_SEARCH}   Adding the Full Search component configuration.
${LISTING_FILTERS}   Adding the Listing Filters component configuration.
${LISTING_PAGING}   Adding the Paging component configuration.
