The Grid data provider class should inherit the Magento\Framework\View\Element\UiComponent\DataProvider\DataProvider class. The collection type of data provider allows us to create already implemented data provider that process the search criteria based on the provided collection class.
Predefined variables will take the following values:
${NAMESPACE}   Created PHP Data Provider class namespace.
${NAME}   PHP class for the Data Provider.
${COLLECTION_CLASS_FQN}   Name of PHP full qualified class name that extends the Magento\Framework\Model\ResourceModel\Db\Collection\AbstractCollection class.
